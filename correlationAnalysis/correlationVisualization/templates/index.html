In 2019 Gregory, Groth, Scharnhorst and Wyatt conducted a survey, resulting in the largest known dataset on how researchers and support professionals discover and use data. A total of 1677 respondents out of 105 countries make up this dataset, working in a wide variety of disciplinary domains. This web application shows the relationship between the disciplinary domains of the respondents and the data types they usually need and how they use those data.

The relationship will be shown with the help of a heatmap. A heatmap is a data visualization technique, where in this case the correlation between the variables is shown in a two dimensional grid, where a more saturated cell represents a higher correlation.

The participants had the freedom to choose more than one domain they work in, together with multiple data needs and uses. In this analysis only participants choosing only one disciplinary domain have been considered, excluding the 4 grouped domains at the end of the list. Here participants only choosing the two named domains have been considered.
These grouped domains have been selected by choosing the top four frequent itemsets using the apriori algorithm.

The chosen data types and needs have been weighted for each user, where each choice had a weight of one divided by the total number of data types and uses each participant chose.

The disciplinary Domain "Social Sciences" has been removed from this list, as no participant has solely chosen this domain.
